# MedContract

Sistema desktop para **gestão de contratos e pagamentos de clientes em clínicas médicas**.

O **MedContract** foi desenvolvido para substituir planilhas e controles manuais por uma solução simples, organizada e visual, permitindo que clínicas acompanhem **clientes ativos, pagamentos mensais, inadimplência e métricas financeiras** em um único sistema.

O projeto foi construído utilizando **Python e PySide6 (Qt)**, com foco em uma interface moderna e intuitiva inspirada em aplicações SaaS.

---

# 🇧🇷 Sobre o projeto

O **MedContract** é um sistema de gestão voltado para clínicas que trabalham com contratos ou mensalidades.

Ele permite registrar clientes, gerenciar dependentes, acompanhar pagamentos e visualizar informações importantes através de um **dashboard centralizado**, facilitando o controle administrativo no dia a dia.

O objetivo é oferecer uma ferramenta leve, rápida e fácil de usar para organização financeira e administrativa.

---

# 🇺🇸 About the project

**MedContract** is a desktop application designed to help clinics manage **client contracts and monthly payments**.

The system centralizes client records, payment tracking and financial indicators in a single dashboard, replacing spreadsheets and manual processes with a more efficient workflow.

It focuses on simplicity, performance and usability for daily administrative tasks.

---

# Funcionalidades

## Gestão de Clientes
- Cadastro de clientes
- Edição e exclusão de registros
- Busca por nome ou CPF
- Controle de status (ativo / inativo)

## Dependentes
- Cadastro de dependentes vinculados ao cliente
- Visualização dentro do perfil do cliente

## Pagamentos
- Registro de pagamentos mensais
- Controle de mês de referência
- Atualização automática do status de pagamento
- Identificação de clientes em atraso

## Dashboard
Painel com informações importantes:

- Total de clientes
- Receita mensal
- Pagamentos realizados no mês
- Estimativa de inadimplência
- Contratos fechados no mês
- Gráfico de receita mensal

## Exportação de dados
- Exportação de clientes
- Exportação de inadimplentes
- Exportação de pagamentos do mês
- Arquivos gerados em planilhas

## Segurança
- Sistema de login
- Níveis de acesso
- Senhas criptografadas com bcrypt

## Backup
- Backup automático do banco de dados
- Armazenamento em pasta do usuário
